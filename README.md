## git add
- 사용법 : git add <추가할 경로>
## git commit -m
- 사용법 : git commit -m "<메시지>"
  "" 따옴표 안써도 됨
  "하고 엔터친다음에 다시 <"> 이거하면 여러줄로 입력됨
  ```
  git commit -m "여러줄
  dquote>
  dquote> 첫번째줄
  dquote> 두번째줄
  dquote> 세번째줄```

## git diff
- 이전 기록(commit)을 기준으롭 변경사항을 본다.
